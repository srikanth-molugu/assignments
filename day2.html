9 HTML Combined Task Challenges

Each challenge includes two distinct tasks—designed to provide varied and comprehensive HTML practice.

1. Sign-Up Form + Art Gallery

  • Create a job application form with radio buttons for gender, dropdown for position, checkboxes for skills, and textarea for motivation.
  • Create an art gallery with 6 images in a grid using <table> and add <figcaption> to each.

2. Blog Article + Travel Brochure

  • Write a short blog article with a heading and a few paragraphs using <article> and <hr>.
  • Design a travel brochure using <div>, <img>, <blockquote>, and <figure> with captions.

3. Conference Schedule + Grocery List

  • Design a conference schedule using <table>, <th>, <caption>, and use colspan for session spanning.
  • Make a grocery list using <ul> and <li> items.

4. Author Biography + Motivational Quote

  • Write a brief biography of a famous author using <section>, <abbr>, <mark>, and organize it within a styled container.
  • Display a motivational quote using <blockquote> and attribute the author with <cite>.

5. Vacation Memories + Book Chapter Outline

  • Create a chapter outline using ordered list with nested unordered lists and highlight chapter titles with <b> or <u>.
  • Write about a recent vacation using <h2>, <p>, and <br> tags.

6. Navigation Menu + Favorite Books Table

  • Add a navigation menu using <nav>, <a>, and internal anchor links (e.g., #about, #contact).
  • Build a table with your top 3 favorite books and authors using <table>, <tr>, <td>.

7. Music Album + Pet Photo

  • Create a music album preview with <audio>, multiple <source> tags, and styled song titles using <strong> or <em>.
  • Insert a photo of your favorite pet with an alt description using <img>.

8. Historical Timeline + Movie Trailer

  • Create a historical timeline of a famous person using <ol>, <time>, and nested <ul> for related events.
  • Embed a YouTube trailer of your favorite movie using <iframe>.

9. Personal Introduction + Chapter Summary

  • Create a personal intro with name, age, and a fun fact using <h1>, <p>, and <br>.
  • Write a summary of your favorite book chapter using <p>, <q>, and <em> tags.
